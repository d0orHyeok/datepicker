# 기본 예제

### 기본 설정

<DatePicker />

<br />
<br />

```javascript
new SnowDatePicker('#element');
```

### 항상 표시

`inline` 옵션을 `true`로 설정하여 달력을 항상 표시할 수 있습니다.

<DatePicker options={{ inline: true }} />

<br />

```javascript
new SnowDatePicker('#element', { inline: true });
```

### 기본 선택된 날짜

`selectedDate` 옵션에 값을 전달하여 기본 선택된 날짜를 설정할 수 있습니다.

<DatePicker options={{ selectedDate: new Date() }} />

<br />
<br />

```javascript
new SnowDatePicker('#element', { selectedDate: new Date() });
```

### 달/연도 선택

`minView` 옵션을 `months | years`로 설정하여 달/연도 선택기를 활성화할 수 있습니다.

<Tabs>
  <TabItem value="monthview" label="달 선택기" default>
    
<DatePicker options={{ minView: 'months' }} />

<br />
<br />

```javascript
new SnowDatePicker('#element', { minView: 'months' });
```

  </TabItem>
  <TabItem value="yearview" label="연도 선택기">
    
<DatePicker options={{ minView: 'years' }} />

<br />
<br />

```javascript
new SnowDatePicker('#element', { minView: 'years' });
```

  </TabItem>
</Tabs>
