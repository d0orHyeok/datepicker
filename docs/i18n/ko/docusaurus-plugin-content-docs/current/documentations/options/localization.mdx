# 현지화

Snow DatePicker는 현지화를 지원합니다. `locale` 옵션에 객체를 전달하여 달력의 언어를 변경할 수 있습니다. 사용 가능한 옵션은 'locales/' 디렉토리에 있습니다.

#### Example of localization object

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
  <TabItem value="localeEnglish" label="English" default>

```javascript
export default {
  name: 'en',
  weekdays: [
    'Sunday',
    'Monday',

    'Thursday',

};
```
