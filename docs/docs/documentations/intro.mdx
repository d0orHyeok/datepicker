# Getting Started

## Install the package

Snow DatePicker is avaialbe as snow-datepicker npm package.

```bash npm2yarn
npm install snow-datepicker
```

## Usage

```js
import SnowDatePicker from 'snow-datepicker';
import 'snow-datepicker/snow-datepicker.css';

new SnowDatePicker('#element'[, options]);
```

### Without package manager

If you don't want ot use package manager, you can add Javascript at the end of `<body>`

```html
<script src="https://cdn.jsdelivr.net/npm/snow-datepicker/snow-datepicker.min.js"></script>
<script>
  new SnowDatePicker('#element'[, options]);
</script>
```
