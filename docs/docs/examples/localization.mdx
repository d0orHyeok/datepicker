# Localization

You can localize the date picker by setting the `locale` option.

### Use predefined locales

Snow DatePicker provides predefined locales for the following languages:
[locale.json](https://github.com/janghye0k/snow-datepicker/blob/main/src/locale.json)

<DatePicker options={{ locale: localeKo, inline: true }} />

<br />
<br />

```javascript
import SnowDatePicker from 'snow-datepicker';
import localeKo from 'snow-datepicker/locale/ko';

new SnowDatePicker('#element', { locale: localeKo, inline: true });
```

Detail of the locale object is described in the [localization](../documentations/options/localization.mdx) section.
